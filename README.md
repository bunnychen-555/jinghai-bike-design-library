# 静海海外自行车设计资料库

电脑端 V2 预览版。当前包含：

- 艺术化首页画廊
- 标签筛选与关键词搜索
- 设计案例详情弹窗
- 三条示例自行车设计资料

## 在线发布

仓库上传完成后，在 GitHub 仓库中进入 `Settings → Pages`，选择：

- Source：`Deploy from a branch`
- Branch：`main` / `(root)`

保存后等待约 1–3 分钟，即可获得公开网址。

## 下一阶段

Google Sheets 将作为内容管理后台，GitHub Actions 定时运行采集程序；新资料先进入待审核状态，审核通过后再更新网站。
